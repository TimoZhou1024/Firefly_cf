## 推荐使用bleachbit清理缓存
安装方法：
```bash
sudo apt install bleachbit
```

使用方法：
```bash
bleachbit
```

使用mobaXterm等工具连接到ubuntu服务器后，直接输入`bleachbit`即可打开图形界面。经测试，清除了我wsl中40个G的垃圾，D盘终于不红了！
